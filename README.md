# Newsfeed

Ths is a news feed Java web application, demonstrating the basics of Spring MVC and Hibernate.

## Features

* News feed view.
* News view by category.
* News search by keywords in title and body.
* Creating and editing of news.
* Deleting news.

## Used technologies

* Spring 4.3.5
* Spring MVC 4.3.5
* Hibernate 5.2.6
* MySQL 5.5.38
* Maven 3.0.3
* Java 1.8
